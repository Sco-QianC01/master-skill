公务员考试培训行业的信息源结构有几个区别于其他培训赛道的特征, 必须在 manifest 之前说清楚。第一, 公考的一手权威信息严格集中在 两条官方管道: 国家公务员局 (scs.gov.cn) 负责国考公告 / 大纲 / 职位表 / 成绩查询的全流程信息发布, 中国人事考试网 (cpta.com.cn) 负责各类职业资格考试报名与成绩查询; 省考信息则分散在 31 个省级人力资源和社会保障厅 / 公务员局官网, 没有统一入口。第二, 考试真题 不由官方公开发布 (考后不发标准答案), 全行业依赖 "考生回忆版" 真题 + 培训机构教研团队整理解析, 这意味着真题数据库天然是 二手 primary (surrogate), 没有任何 verified_primary 的官方真题全文库。第三, 培训行业 三巨头 (中公教育 002607.SZ / 华图山鼎 300492.SZ / 粉笔 02469.HK) 均为上市公司, 其 财报 / 招股书 / 年报 是理解行业规模 / 渗透率 / 竞争格局的 一手数据 (上市公司公告属 verified_primary); 第三方行业报告 (华经产业研究院 / 观研报告网 / 前瞻产业研究院) 是二手加工。第四, 公考备考社区 高度活跃但分散: QZZN 论坛 (bbs.qzzn.com, 461 万注册用户 / 1161 万帖) 是最大垂直论坛, 豆瓣考公小组 / 小红书考公圈 / B 站公考名师频道 形成多平台矩阵, 但它们的内容属 UGC reference 而非 primary。第五, 播客 / 音频在公考领域以 "伴学" 和 "体制内生活分享" 为主, 真正聚焦 应试技术 的高质量播客极少, 更多作为备考心态 / 职业认知的补充信源。第六, 黑名单规则: 知乎 / 微信公众号 (mp.weixin.qq.com) / 百度百科 / 百度知道 / CSDN 不入 manifest, 尽管它们是中文公考话题的搜索默认入口 -- 替代路径是去对应主体的官网 / 上市公司公告原件 / 论坛原帖。

### 一手率说明
本行业一手信息的特殊性: (a) 考试真题没有官方全文发布渠道, 全行业依赖考生回忆版, 天然 surrogate; (b) 培训机构的教研内容 (课程设计 / 教学方法 / 命题预测) 是行业核心知识, 一手来源是机构官网和上市公司公告; (c) 政策信息高度集中在 .gov.cn 域名, 一手可验证; (d) 社区内容 (QZZN / 豆瓣 / B 站) 是理解行业生态的重要维度, 但按规则标 reference。
