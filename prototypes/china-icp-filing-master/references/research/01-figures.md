# Track 01 — Figures: China ICP Filing & Internet Content Compliance

中国网站备案与互联网内容合规这条线的 figures 分布相比海外科技政策圈反常地 **结构清晰**, 但同时 **个体可定位性不均衡** —— 这种不对称性必须在 manifest 之前讲清楚, 否则后续蒸馏会把不同类型的 figure 错位混用. 第一, **学术轨**是这条线最有名字的轨道, 因为 PIPL / DSL / CSL 三法的立法专家组成员名单本身就是公开的, 且这些专家几乎全部集中在 5 所院校 (中国社科院法学所 / 人大法学院 / 清华法学院 / 北大法学院 / 华东政法), 学者主页都在 `.edu.cn`, 著作在 法律出版社 / 中国民主法制出版社 publisher listing 公开可查, 一手率高得反常. 第二, **律所合伙人轨**是 实务执行层 唯一公开个体 figure 的轨道, 红圈律所 (君合 / 中伦 / 金杜 / 大成 / 海问 / 汉坤 / 锦天城) 的网络与数据合规业务线合伙人都在自家律所官网团队页公示, alerts 文章署名公开, 但要小心 微信公众号搬运的合伙人发言不可引 —— 必须回律所官网原文. 第三, **监管轨 figure 高度匿名化** —— 工信部信管局 / 网信办网络数据管理局 / 公安部网安局的现任司局领导很少做署名公开演讲, 政策解读多以"部委发言人"或"司局负责人"非具名形式公布, 这是中国政府公务员职务行为非个体化的客观结构, 因此本表只能用"XX 部司局领导 / 退休司长 + 解读专家"的脱敏方式收录, 不试图强行实名. 第四, **行业组织 figure** 主要是协会会长 / 研究院院长级别 (CAICT 信通院 / ISC 互联网协会 / CIE 电子学会), 但具体研究员个体较少公开 fig 化, 多以机构身份发声. 第五, **反面案例** 选取必须严格区分"案例视角"与"个人攻击" —— 滴滴 80.26 亿罚款案、知网反垄断案、字节算法处罚案是这条线无法绕开的历史样本, 但都以 **企业主体** 而非个人 figure 形式收录, 引 CAC / 工信部 / 国家市场监督管理总局 一手通报, 不引第三方报道. 第六, **percentages 必带 caveat** —— 行业内大量"影响 X% 互联网公司""覆盖 X% 大模型"这类数字往往是律所 alerts 或协会公开估算, 缺乏严格统计基础, 凡出现必须 inline 标注来源等级.
