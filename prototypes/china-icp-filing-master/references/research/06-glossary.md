# Track 06 — China ICP Filing & Internet Content Compliance Glossary

中国网站备案 (ICP filing) 与互联网内容合规 是一个 **法规简称密集 / 一手主体一手率天然高** 的行业 — 几乎所有核心术语都直接来自 国务院令 / 部委规章 / 国家标准 / 监管机构 公告, 没有"业界俗称取代法定术语"的灰色地带, 因此 verified_primary (`.gov.cn` / `.org.cn` / `.edu.cn`) 来源率远高于一般技术行业。然而, 该领域的 **独特性陷阱** 在于 中国互联网合规并非"一张牌照打天下", 而是 **多重独立行政许可 / 备案体系平行叠加**, 一个看似简单的 SaaS 网站可能同时触发: (1) **ICP 备案** (工信部, 信部令 33 号, 非经营性, 接入商代办), (2) **ICP 经营许可证** (工信部, B25 信息服务业务, 经营性, 注册资本 100w/1000w), (3) **公安联网备案** (公安部, 公网安备号, 30 日内办理), (4) **APP 备案** (工信部 2024-03 新规, 移动应用单独备案), (5) **算法备案** (网信办, 算法推荐 / 深度合成 / 生成式 AI 三套独立备案), (6) **等保测评** (公安部, GB/T 22239-2019, 三级以上须公安备案), (7) **数据出境** (CAC 评估 / 标准合同 / 认证三选一), (8) **大模型服务备案** (网信办, 生成式 AI 暂行办法). 外加 监管机构 在 "工信部 (MIIT) / 网信办 (CAC) / 公安部 / 市监总局 / SAC TC260" 之间存在 **职能交叉**, 同一事项常有多家联合执法. 增值电信业务分类 (B11/B12/B13/B14/B22/B25/B26) 是 工信部 2015 年修订的 《电信业务分类目录》 法定分类, 直接决定经营许可证类型与注册资本门槛. 等保 2.0 (GB/T 22239-2019) 五级分级 与 关基条例 CII 是两套不同体系, 常被混淆. 本表覆盖 95+ 条术语, 跨备案核心 / 法规简称 / 监管机构 / 业务分类 / 等保 / 数据 / 算法 / 实务流程 八大类, 法规原文优先引 国务院 / 工信部 / 网信办 / 公安部 .gov.cn 入口, 标准引 openstd.samr.gov.cn (国家标准全文公开系统), 律所 / 智库 / vendor docs 在 surrogate_primary bucket 并附 literal 关键词标注.
