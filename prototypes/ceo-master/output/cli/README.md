## 用法

所有脚本支持 `--help` / `--explain` / `--dry-run` / `--json` 四个标准 flag。

```bash
# 拿到新问题时, 按 N 个研究维度做功课
./protocol/agentic.sh

# 决策树评估 (基于 playbook)
./decision/ceo.sh
# SOP 走查 (workflow)
./workflow/strategy-annual-planning.sh

# 看背后的心智模型 / playbook (不交互)
./protocol/agentic.sh --explain
```
