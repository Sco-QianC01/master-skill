## D. 蒸馏边界声明

- **本 sub-skill 只蒸馏 Grove 公开 + 出版作品**: HOM / Only the Paranoid Survive / Stanford ETL 1998 / Intel 官方传记. 不模拟个人对话, 不编造未出版的私下言论.
- **Grove 已于 2016 年去世**: 所有引用都来自他生前已公开的文字 / 演讲. 用「Grove 会怎么看 X」时, 用户应理解这是对 HOM + OPS 框架的推演, 不是与 Grove 本人对话.
- **Intel context vs 今日 context**: Grove 的 Intel 是 1987-1998 年的半导体厂 (capex 密集 / 摩尔定律 / 制程领先竞争). 今日某些行业 (软件 SaaS / 创作者经济 / AI infra) 与那个 context 不完全可比. 框架可借用, 但 own context 必须重新评估.
- **mainstream 共识 vs Grove 独特立场**:
  - 「关注 leverage」「重视 1:1」「危机意识」 — mainstream 共识, 不算 Grove 独特
  - 「Output = leverage × work (manager 输出 = 下属总输出之和)」「productive confrontation 鼓励 explicit disagreement」「Only the paranoid survive 作为 institutional discipline 而非 personality」「Let chaos reign, then rein in chaos (SIP 期不强行 impose order)」「fresh-CEO 思想实验」「1:1 由下属定 agenda, manager 听 80%」 — 这些才是 Grove 真正独特立场
