# Track 05 — Sources (implant dentistry / 口腔种植)

> Wave 1 Track 05 — newsletter / podcast / conference / society / journal / education-platform / 中国学会与大学医院. Industry: implant dentistry, locale=global (中外双语并重, profile=practitioner).
>
> 结构约束 notes (诚实):
> (1) 英文一手信源密度远高于中文 (8-15x): COIR / JOMI / IJPRD / J Periodontol / JCP / JPD / J Esthet Restor Dent / Clin Implant Dent Relat Res / Periodontology 2000 + ITI / AO / EAO / EFP / AAP 共识 + Cochrane Esposito 系列 系统综述, 是 evidence-based implantology 实质骨架.
> (2) 中文一手优质内容 大量被困在 公众号 (mp.weixin.qq.com 黑名单): 林野 / 宿玉成 / 邱立新 / 满毅 / 周磊 / 廖贵清 等 KOL 的 临床讲座 / 病例分享 主战场是 公众号 + 医院内部, 我们改走 中华口腔医学会 (cma.org.cn) + 中华口腔医学杂志 + 中华口腔种植学杂志 + 大学医院教师页 (kq.bjmu.edu.cn 北医口腔 / kq.scu.edu.cn 华西口腔 / 9hospital.com.cn 上海九院 / shckq.com 中山光华) 替代.
> (3) Eur J Oral Implantol (Esposito-edited) 2019 停刊, 历史 archive 价值仍在 Quintessence (quintpub.com).
> (4) 厂商赞助 RCT bias: 教科书 + Cochrane Esposito 系统综述权重高于 vendor 单中心 RCT — 但 vendor 临床手册 (Straumann/Nobel/Geistlich/BioHorizons) 作为 surrogate_primary 仍是产品规格 + 注册证号 + 适应症 一手出处.
> (5) Podcast: 口腔种植类英文 podcast 数量远少于通用编程, 但 The Implant Show (Yiming Shi, Australia) / Talking Implants (UK) / IDEA / Inside Dental Technology Podcast 等近 5 年已成型; 中文目前主流 podcast 稀少, 用 学会 webinar (ITI Academy 中文化 / 中华口腔医学会 直播) 替代.
> (6) 教育平台 (Spear / gIDE / dentalXP / ITI Academy / Osseo Academy) 是 实战 lecture / 课程 一手, 与 期刊 evidence 互补 — 标 `surrogate_primary` + note `syllabus` / `lecture` / `课程`.
> (7) Dentaltown forum 大型行业社区, 临床 case 讨论密度高, 标 `reference` (社区论坛非一手).
> (8) 反例 / 临床警示 (peri-implantitis / All-on-4 适应症泛化 / 即刻负重失败 / 美学失败 / BRONJ) 全引一手 published case + EFP/AAP 共识立场, 不软化, 不指名个人.
